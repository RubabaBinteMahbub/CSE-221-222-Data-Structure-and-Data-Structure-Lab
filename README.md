### CSE-221-222-Data-Structure-and-Data-Structure-Lab
This repository serves as a comprehensive collection of Data Structure implementations from the CSE-221 (Theory) and CSE-222 (Lab) courses. It focuses on the architectural design, memory allocation, and management of data.

🚀 Course Overview
The primary objective of this repository is to demonstrate how different data formats are stored in memory and how their relationships are maintained.

Key areas include:
Time and Space Complexity: Big O, Big Omega, and Big Theta notations.

Linear Data Structures: Arrays, Linked Lists, Stacks, and Queues.

Non-Linear Data Structures: Trees, Graphs, and Heaps.

Sorting & Searching: Quick Sort, Merge Sort, Binary Search, etc.
