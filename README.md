### CSE-221-222-Data-Structure-and-Data-Structure-Lab
This repository serves as a comprehensive collection of Data Structure implementations from the CSE-221 (Theory) and CSE-222 (Lab) courses. It focuses on the architectural design, memory allocation, and management of data.

<h3>🚀 Course Overview</h3>
<b>Course Code:</b> CSE-221 (Theory) | CSE-222 (Lab)
<br>
<b>Language Used:</b> C++ (Standard: C++20)
<br>
<b>Objective:</b>The primary objective of this repository is to demonstrate how different data formats are stored in memory and how their relationships are maintained.
<br><br>
<b>Key areas include:</b>
<br>
<b>Time and Space Complexity:</b> Big O, Big Omega, and Big Theta notations.

<b>Linear Data Structures:</b> Arrays, Linked Lists, Stacks, and Queues.

<b>Non-Linear Data Structures:</b> Trees, Graphs, and Heaps.

<b>Sorting & Searching:</b> Quick Sort, Merge Sort, Binary Search, etc.

<h3>🛠 Setup & Usage</h3>
<b>Compiler :</b> Use any standard C compiler (GCC, Clang, or IDEs like CodeBlocks/VS Code).
